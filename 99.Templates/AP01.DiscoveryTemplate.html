üîß Query Pattern Examples

1. Date Function Usage

Before: INSERT INTO SAMPLE_TABLE (..., CREATE_DATE, UPDATE_DATE) VALUES (..., CURRENT_DATE, CURRENT_DATE) UPDATE SAMPLE_TABLE SET UPDATE_DATE = CURRENT_DATE WHERE ...
After: INSERT INTO SAMPLE_TABLE (..., CREATE_DATE, UPDATE_DATE) VALUES (..., NOW(), NOW()) UPDATE SAMPLE_TABLE SET UPDATE_DATE = NOW() WHERE ...

2. System Table Optimization

Before: SELECT #{empno} EMPNO FROM SYSTEM_TABLE SELECT 'ID'||TO_CHAR(CURRENT_DATE, 'YYMMDD') || LPAD(SEQUENCE_NAME.NEXTVAL, 4, '0') ID_NO FROM SYSTEM_TABLE
After: SELECT #{empno} AS EMPNO SELECT 'ID'||TO_CHAR(NOW(), 'YYMMDD') || LPAD(NEXTVAL('SEQUENCE_NAME'), 4, '0') AS ID_NO

3. Null Handling Functions

Before: SELECT NVL((SELECT S.USER_NAME FROM USER_TABLE S WHERE S.USER_ID = A.CREATOR_ID AND ROWNUM = 1), A.CREATOR_ID) CREATOR_NAME
After: SELECT COALESCE((SELECT S.USER_NAME FROM USER_TABLE S WHERE S.USER_ID = A.CREATOR_ID LIMIT 1), A.CREATOR_ID) AS CREATOR_NAME
